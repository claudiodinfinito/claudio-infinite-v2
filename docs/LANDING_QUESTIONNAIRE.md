## 🎯 Checklist Pre-Lanzamiento (para el agente)

Antes de entregar la landing, verificar:

- [ ] Hero tiene headline que comunica UVP claramente
- [ ] Beneficios > Features (qué ganan, no qué hace)
- [ ] CTA visible arriba del fold (sin scroll)
- [ ] Testimonios específicos con nombre/foto
- [ ] Urgencia/escasez genuina (no falsa)
- [ ] Formulario simple (mínimo campos)
- [ ] Mobile-first design
- [ ] Velocidad < 3 segundos
- [ ] Tracking instalado (GA4, Pixel, etc.)
- [ ] A/B test planeado (mínimo 2 headlines)
